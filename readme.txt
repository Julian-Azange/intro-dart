## Etiquetas para Identificar Respuestas e Impresiones

Dentro del código plasmado en cada archivo, se han definido prefijos o "etiquetas" con el objetivo de identificar de forma clara cada una de las respuestas o impresiones por consola.

A continuación se presentan los prefijos definidos y utilizados en el desarrollo de cada ejercicio:

INFO: 'Informacion'
ERROR: 'Error'
WARNING: 'Advertencia'
DEBUG: 'Depuracion'
SUCCESS: 'Exito'
FAIL: 'Fallo'
ALERT: 'Alerta'
NOTICE: 'Aviso'
CRITICAL: 'Critico'
EXCEPTION: 'Excepcion'
TRACE: 'Rastreo'
FATAL: 'Fatal'
IMPORTANT: 'Importante'
UPDATE: 'Actualizacion'
RESULT: 'Resultado'
CONFIG: 'Configuracion'
STATUS: 'Estado'
LOG: 'Registro'
VERBOSE: 'Detallado'
IMPORTANT: 'Importante'
INIT: 'Inicializacion'
FINALIZE: 'Finalizacion'
VALIDATE: 'Validacion'
START: 'Inicio'
STOP: 'Detener'
OK: 'OK'
ACCEPTED: 'Aceptado'
VALID: 'Valido'
COMPLETED: 'Completado'
FOUND: 'Encontrado'
FULFILLED: 'Cumplido'
APPROVED: 'Aprobado'

Estas etiquetas te permitirán identificar fácilmente la naturaleza de cada mensaje o respuesta en la consola, brindando mayor claridad y comprensión en el flujo de tu aplicación.
